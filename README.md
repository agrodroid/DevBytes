# DevBytes
Android app that lists a number of videos by Google for Android developers.
## Authors

- [@agrodroid](https://github.com/agrodroid)

  
## Features

- 100% Kotlin
- Fully using MVVM pattern
- Videos open in YouTube
- Local caching using Room Library
- Uses Retrofit + Moshi
